# Ladybird

[Ladybird](https://ladybird.org) is a truly independent web browser, using a novel engine based on web standards.

These are unoffical binaries are are not endorced by the team. 

This is just the code ran through Github Action [because I had issues building on Gentoo](https://github.com/LadybirdBrowser/ladybird/issues/3836).

# Install

Download artifact - https://github.com/unofficial-ladybird-builds/ladybird-bin/actions



``mkdir Ladybird``



``unzip ladybird-full-repo.zip``

``unzip ladybird-full-repo.zip`` (Yes, two times)


``./Build/release/bin/Ladybird``
